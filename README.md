# node-image2geotiff-sample

## Dependency
* GDAL (Geospatial Data Abstraction Library) : 공간처리 Tool 
* PROJ : GIS 좌표계 System
* Bash Shell   

### link
  * OSGeo4W (for windows): https://trac.osgeo.org/osgeo4w/
  * GDAL : https://gdal.org/download.html
  * PROJ : https://proj.org/install.html

***

## Warning
* MacOS에서만 테스트 되었음
* Windows에서는 GDAL 사용을 위해 Python 이나 Linux 환경 세팅이 필요함
